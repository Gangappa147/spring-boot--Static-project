Doctor-Chat Bot

• I developed a doctor Chatbot using an LSTM model for natural language processing.

• I worked on data preprocessing tasks such as tokenization, stemming, and vectorization to prepare the dataset.

• The chatbot offers health condition insights, recommendations, and disease-related information, providing users with reliable, AI-driven medical assistance and advice.

Heart Disease Prediction: -

• I Developed a machine learning model to analyze a dataset of 500 patient records, achieving an accuracy of 85 percentage in detecting heart disease.

• The aim of this project is to create a machine learning model for predicting heart disease by analyzing patient data.

• This tool will help healthcare professionals identify at-risk individuals early.
